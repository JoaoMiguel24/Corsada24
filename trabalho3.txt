Segunda linguagem: Java - Uma linguagem de programação de propósito geral, Conhecida por sua portabilidade, pois o código Java pode ser executado em diferentes plataformas sem a necessidade de recompilação. Usada em uma variedade de aplicativos, desde aplicações empresariais até desenvolvimento de aplicativos Android.
Código mínimo:
import java.util.*;
import java.lang.*;
import java.io.*;

// The main method must be in a class named "Main".
class Main {
    public static void main(String[] args) {
        System.out.println("Bem vindo a programação!");
    }
}

Terceira: C++ - Uma linguagem derivada da linguagem C. Ela é conhecido por sua flexibilidade e eficiência, sendo escolhido em situações em que o desempenho é crítico e o controle sobre os recursos do sistema é necessário.
