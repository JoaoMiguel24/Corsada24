Exemplo de uma estrutura de repetição no código python:

# Exemplo de estrutura de repetição for
for i in range(5):
    print(i)

# Exemplo de estrutura de repetição while
contador = 0
while contador < 5:
    print(contador)
    contador += 1

Exemplo de uma estrutura de tomada de decisão no código python:

# Estrutura de tomada de decisão para aprovação ou reprovação
nota = 7

# Verifica se o aluno foi aprovado, reprovado ou precisa de recuperação
if nota >= 7.0:
    resultado = "Aprovado!"
elif 5.0 <= nota < 7.0:
    resultado = "Recuperação."
else:
    resultado = "Reprovado."

# Exibe o resultado
print(f"Resultado: {resultado}")
